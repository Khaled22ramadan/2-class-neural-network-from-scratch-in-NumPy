Deep Learning · Week 3

Planar Data Classification
One Hidden Layer Neural Network

Input → Output flowchart for each stage

STAGE 01

Load & Explore Dataset

load_planar_dataset()
INPUT
None (built-in flower dataset)
Auto-generated 2D "flower" pattern with red/blue points
→
OUTPUT
X — shape (2, 400) Y — shape (1, 400)
X = 2 features (x1, x2) · Y = binary labels (0=red, 1=blue) · m=400 examples
STAGE 02

Baseline — Logistic Regression

sklearn.LogisticRegressionCV
INPUT
X — (2, 400) Y — (1, 400)
Same flower dataset — testing if a linear boundary suffices
→
OUTPUT
Accuracy ≈ 47% Conclusion: not linearly separable
Logistic regression fails → need non-linear model (neural network)
STAGE 03

Define Network Architecture

layer_sizes(X, Y)
INPUT
X — (2, 400) Y — (1, 400)
Shape of data matrices determine layer sizes
→
OUTPUT
n_x = 2 (input features) n_h = 4 (hidden units) n_y = 1 (output)
Architecture: 2 → 4 → 1 neurons
STAGE 04

Initialize Parameters

initialize_parameters(n_x, n_h, n_y)
INPUT
n_x = 2 n_h = 4 n_y = 1
→
OUTPUT · params dict
W1 — (4, 2) random × 0.01 b1 — (4, 1) zeros W2 — (1, 4) random × 0.01 b2 — (1, 1) zeros
Training Loop — repeat for N iterations
STAGE 05

Forward Propagation

forward_propagation(X, parameters)
INPUT
X — (2, 400) W1, b1, W2, b2
Data + current weights
→
OUTPUT
Z1 = W1·X + b1 — (4, 400) A1 = tanh(Z1) — (4, 400) Z2 = W2·A1 + b2 — (1, 400) A2 = σ(Z2) — (1, 400) ← ŷ cache: Z1, A1, W1, W2
STAGE 06

Compute Cost

compute_cost(A2, Y)
INPUT
A2 — (1, 400) predictions Y — (1, 400) true labels
→
OUTPUT
J — scalar (cross-entropy loss)
J = −(1/m) Σ [ y·log(ŷ) + (1−y)·log(1−ŷ) ]
STAGE 07

Backward Propagation

backward_propagation(params, cache, X, Y)
INPUT
params — W1, b1, W2, b2 cache — Z1, A1, W1, W2 X — (2, 400) Y — (1, 400)
→
OUTPUT · grads dict
dW1 — (4, 2) db1 — (4, 1) dW2 — (1, 4) db2 — (1, 1)
Gradients via chain rule (backprop through tanh + sigmoid)
STAGE 08

Update Parameters

update_parameters(params, grads, learning_rate)
INPUT
params — current W1,b1,W2,b2 grads — dW1,db1,dW2,db2 learning_rate α = 1.2
→
OUTPUT
W1 ← W1 − α·dW1 b1 ← b1 − α·db1 W2 ← W2 − α·dW2 b2 ← b2 − α·db2
Updated parameters fed back into Stage 05 for next iteration
STAGE 09

Predict & Evaluate

predict(parameters, X)
INPUT
trained parameters W1,b1,W2,b2 X — (2, 400)
→
OUTPUT
predictions — (1, 400) binary {0,1} Accuracy ≈ 90% Decision boundary plot
A2 > 0.5 → 1, else → 0 · Dramatic improvement vs. 47% baseline
